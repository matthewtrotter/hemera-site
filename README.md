# Hemera Engineering Services LLC Website

Static GitHub Pages website for Hemera Engineering Services LLC.

## Pages Setup

This repository can be published directly with GitHub Pages:

1. Open the repository settings on GitHub.
2. Go to **Pages**.
3. Set the source to the `main` branch and the repository root.
4. Save the settings.

## Custom Domain

After registering a domain in Cloudflare:

1. Add a `CNAME` file at the repository root containing the domain name.
2. Configure the domain in GitHub Pages.
3. Add the required DNS records in Cloudflare.
4. Enable HTTPS once GitHub Pages finishes provisioning the certificate.
